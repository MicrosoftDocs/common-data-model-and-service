# Data Format

An enum class containing the data formats. 

```csharp
public enum CdmDataFormat
{
    Unknown,
    Int16,
    Int32,
    Int64,
    Float,
    Double,
    Guid,
    String,
    Char,
    Byte,
    Binary,
    Time,
    Date,
    DateTime,
    DateTimeOffset,
    Boolean,
    Decimal,
    Json
}
```
